# spring-boot-mongodb-react

## `Backend`

#### Installation Dependencies

```bash
$ mvn install
```

#### Configuration

Change you config in [application.properties](./backend/src/main/resources/application.properties)

#### Run

```bash
$ mvn spring-boot:run
```

## `Frontend`

#### Installation Dependencies

```bash
$ npm install
```

#### Run

```bash
$ npm build 
$ npm start
```
